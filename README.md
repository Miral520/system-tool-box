# vue_electron

### 介绍
##### 结合Electron和Vue开发PC端和Mac端软件<br/><br/>

### 安装
```javascript
npm install
```
<br/>

### 启动
##### 需要新建两个终端<br/><br/>
##### 终端1，首先运行
```javascript
npm run serve
```
<br/>

##### 终端2，再运行
```javascript
npm run win // 开发模式

npm run app // 或者生产模式，无devTool
```
<br/>

### 编译
##### 暂时为Vue编译<br/><br/>
```javascript
npm run build
```
<br/>
